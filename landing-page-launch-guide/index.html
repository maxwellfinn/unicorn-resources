üîß GoDaddy

  1. Log into godaddy.com
  2. My Products ‚Üí Find domain ‚Üí DNS
  3. Click ADD
  4. Type: CNAME | Name: subdomain | Value: your URL
  5. Save

üîß Namecheap

  1. Log into namecheap.com
  2. Domain List ‚Üí Manage ‚Üí Advanced DNS
  3. ADD NEW RECORD
  4. Type: CNAME | Host: subdomain | Target: your URL
  5. Click ‚úì to save

üîß Cloudflare

  1. Log into dash.cloudflare.com
  2. Click domain ‚Üí DNS
  3. Add record
  4. Type: CNAME | Name: subdomain | Target: your URL
  5. ‚ö†Ô∏è Click orange cloud to turn GRAY
  6. Save
‚ö†Ô∏è
Critical: Cloud must be GRAY (DNS only), not orange!

üõí Shopify-Managed Domains

  1. Log into Shopify Admin ‚Üí Settings (bottom left)
  2. Click Domains
  3. Click the domain you want to modify
  4. Scroll to Subdomains section
  5. Click Add subdomain
  6. Enter subdomain prefix (e.g., promo for promo.store.com)
  7. Select Point to external website
  8. Enter your Netlify/Cloudflare URL as the target
  9. Click Save
üí°
Domain Managed Elsewhere? If the domain was purchased outside Shopify (GoDaddy, Namecheap, etc.) but connected to Shopify, the DNS is still managed at the original registrar. Use those instructions instead.
‚ö†Ô∏è
Shopify Limitation: Shopify only supports subdomains for external sites. You cannot point the root domain (store.com) to an external landing page while keeping Shopify active.

üåê DNS Issues

‚ùå DNS_PROBE_FINISHED_NXDOMAIN

DNS hasn't propagated or CNAME wasn't added.

‚úÖ Wait 15-30 min. Verify CNAME values are correct.

‚ùå Client doesn't know DNS login

Can't access domain registrar.

‚úÖ Check email for "domain registration" receipts. Try password reset. Use whois.domaintools.com

‚ùå Client entered full domain

Typed promo.example.com instead of just promo.

‚úÖ Name field should ONLY contain the subdomain part.

‚ùå Conflicting record exists

Already an A record for same subdomain.

‚úÖ Delete existing record first, then add CNAME.
üîí SSL Issues

‚ùå SSL Certificate Error

Certificate not issued yet.

‚úÖ Click "Verify DNS" in dashboard. Wait 5-10 min.

‚ùå Mixed content warnings

Page loads HTTP resources.

‚úÖ Change all http:// URLs to https:// in HTML.

‚ùå Redirect loop

Cloudflare + Netlify SSL conflict.

‚úÖ Set Cloudflare SSL to "Full (strict)", not "Flexible".
üìÅ File Issues

‚ùå 404 Not Found

Main file not named correctly.

‚úÖ Must be named index.html (lowercase) in root folder.

‚ùå Images broken

Wrong paths or missing files.

‚úÖ Use relative paths: assets/images/photo.jpg. No absolute paths.

‚ùå Case sensitivity

Image.PNG vs image.png

‚úÖ Linux servers are case-sensitive. Rename to lowercase.
‚òÅÔ∏è Cloudflare Issues

‚ùå Orange cloud enabled

Should be gray when pointing to Netlify.

‚úÖ Click orange cloud to turn GRAY. Gray = DNS only.

‚ùå SSL mode "Flexible"

Causes redirect loops.

‚úÖ Change to "Full" or "Full (strict)".

‚ùå Caching old content

Changes not showing.

‚úÖ Caching ‚Üí Purge Everything.
üõí Shopify Issues

‚ùå Can't find subdomain settings

Option missing in Shopify admin.

‚úÖ Settings ‚Üí Domains ‚Üí Click domain ‚Üí Scroll to "Subdomains" section. If missing, domain may be managed externally.

‚ùå Domain managed by third-party

Bought domain outside Shopify but connected it.

‚úÖ DNS is at original registrar (GoDaddy, Namecheap, etc). Add CNAME there, not in Shopify.

‚ùå "External website" option missing

Older Shopify plan or settings.

‚úÖ Upgrade plan or contact Shopify support. Alternative: manage DNS at domain registrar directly.

‚ùå Want root domain for landing page

Client wants example.com, not promo.example.com

‚úÖ Not possible while Shopify active. Options: use subdomain, temporarily disconnect Shopify, or host landing page as Shopify page.

‚ùå Subdomain shows Shopify 404

Subdomain pointing to Shopify instead of external.

‚úÖ Delete subdomain in Shopify, add fresh with "External website" option. Or add CNAME at registrar if domain is external.

‚ùå SSL not working on subdomain

Certificate error on the new subdomain.

‚úÖ Shopify doesn't issue SSL for external subdomains. SSL comes from your hosting (Netlify/Cloudflare). Verify DNS in hosting dashboard.
üè¢ Business Blockers

‚ùå "IT handles DNS"

Client doesn't have access.

‚úÖ Prepare clear instructions for IT team. Offer to join a call.

‚ùå Subdomain in use

Already pointing somewhere.

‚úÖ Use different subdomain (offer.site.com) or delete old record.

‚ùå "Just host it for me"

Client doesn't want to deal with DNS.

‚úÖ You're on Path A! Just send them the CNAME to add.
